搭建hexo博客问题汇总

发表于2021-08-01|更新于2021-08-16|博客hexo
|阅读量:

相遇皆是缘分

问题汇总

大家在观看了我 搭建hexo博客系列视频后可能会遇到的一些问题 视频地址

问题一：npm install 报错

解决方法：

【方法一】

在git 界面 依次执行下面命令(如果下面的命令报错了，不用理会，各执行一次即可)

npm clean
 
npm run clean
 
npm  clean-install
 
npm install
 
npm run dev

执行完后，再次执行 

npm install

【方法二】

关闭所有的杀毒软件（包括系统自带的杀毒软件）或者 用管理员权限打开 git 执行命令

问题二：部署不了 github

解决方法：

首先测试，输入下面代码

hexo g && hexo s

如果本地部署可以成功 并且 浏览

  1. 登录github

  2. 查看你的分支下是否有 master (部署后会出现)

  3. 将默认分支修改为 master ，其他的无用分支可以删掉

问题三：音乐显示不出来

解决方法：

音乐建议只使用 网易 和 qq，其他的可能有些问题

问题四：首页背景轮播

解决方法：

建议关掉轮播 在 Blog\themes\matery 文件下打开 _config.yml

ctrl + f 或 ctrl +h 输入“轮播” 二字 来查找， 将true 改为 false即可

问题五：如何管理评论

  1. 先在自己的博客评论区随便写一个评论

  2. 登录 leancloud

  3. 点击 数据存储 - 结构化数据

  4. 所有的类里有一个类 显示 1，表示它存储了一条你刚刚评论的数据

  5. 点击那个类，即可管理你评论区的所有数据
